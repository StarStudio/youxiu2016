电子工程学院2011029100班:守心互助 携梦同行

文：李果  /  图：邝俊  /  来源：新闻中心  /  2015-05-05

　　

来到成电的每个人，心底都描有一幅磅礴的蓝图，在这对无限未来的憧憬里，无处不渗透着年轻一代最珍贵也最易丢失的两个要素——梦想和本心。十班的四年，完整地展现了对这两者的追求和收获。



NEW



你真正渴望的是一个怎样的未来呢？为此需要做什么？你真的清楚自己的能力吗？导生抛出这些问题时，正值我们作为十班这个集体，第一次聚在一起。每个人心底都对这一系列问题产生了思考，青春热血也从高考后的沉眠中被唤醒，我们开始直面自己的内心，思考“梦想”二字的真正指向，也对班级建设有了最初的定位：十班，必须是一个追求梦想的集体。



在辅导员和导师的指导下，大家积极了解学校的政策、提供的资源，关注专业前景和未来发展，设定每个人对未来的目标和实施方案，互相督促学习计划的实行，并在后期及时修正和拓展。同时，在缺失了显性压力的大学生活中，我们共同承诺绝不再因往日的成绩而沾沾自喜，绝不虚度光阴，始终保持专注认真的学习态度。这是十班最大的特色，对学习和科研的热情是促使我们走向成功最大的动力源泉。 那时起，十班就确立了冲刺“邦清班”的目标，并坚持不懈地朝着这个目标努力。

BUILD



十班始终提倡良好的学风，小组自习、学习经验交流、考后分析、成果汇总等促使同学们积极主动学习，也带动了整体的进步。班级在第一年就取得了年级第一的好成绩。



但更重要的是，我们对每一件事都认真对待，并努力去执行。同学们积极参加科研项目和各种竞赛，创新项目评优，数学建模美赛一等奖、省赛一等奖、二等奖、电子设计省二等奖、全国英语竞赛三等奖等都是我们辛勤付出的成果。成电杯辩论赛、篮球赛、足球赛，冠军殊荣，都有我们的一份力；经管双学位、创业策划、电视大赛、短片设计、志愿青奥，课堂之外，我们也有无限风采。在提高人文素养方面，班级创设了流动图书架，平日里自由借阅，分享读书心得。我们还开发了微信公众号“成电传承”，为大家提供科技前沿信息和美文趣事，丰富了班级的课余生活，也为同学施展文学才华提供了很好的平台。党建成绩突出，同学们积极投身实践活动，班干部们也十分负责，每年换届不连任的传统使更多人亲身参与到班级建设中，大家互相体谅，共同为班级活动出谋划策。生活中我们也遵守共同拟定的公约，面对问题也平心静气地商量解决，班级里的每个人都紧密地联系在一起，如同家人一般。



不谦虚的说，十班是一个全面发展的优秀班级，也正如聚会上同学们真挚的表白，我们每个人都为身在十班而骄傲！



DEBUG



然而，每个人都期待成长带来的效益，迫切需要来自周围环境的肯定，这过程中也会因挫折而灰心丧志。而在十班，我们有帮助彼此自省和振作视自己的伙伴。以寝室为单位自发地督促掉队的同学上自习，帮助他跟上课程进度，一起解决难题，成为了十班的一大特色。伙伴间的友情、同学间的良性竞争是一剂良药，能使我们以人为镜看清自己的不足，杜绝虚度光阴的傻事，日日夜夜的共同努力，掉队同学黑马奇袭般的成绩突破，使整个班级欣欣向荣，暂时掉队的同学也重获自信并得到进步。“一带一”和学习小组由此兴起，谁也没有再落下，十班作为一个整体变得越来越优秀，我们也一次又一次超越自己。



而在求认同的过程中，我们也曾忽视自己真实的感受和需要。但十班的同学，总是能够在畅谈理想的时候，轻击敲醒彼此的心灵。我们乐于分享彼此的成长，尤其大三时同学们分享了各自在实习或夏令营中历练的感悟，提出了很多具有深刻思考意义的话题，我们究竟是为了什么而奋斗，我们能给世界留下什么。十班绝不乏才智卓越之人，然而我们更骄傲的是，有这么多个性鲜明、积极对待人生的伙伴，我们所向往的并不只是个人的成功，更是对事业、家庭、社会的贡献。



在这冲向梦想的竞赛中，也有人于困顿中发现了自己隐藏的力量转而选择了另一条与众不同的道路，十班的同学也衷心地祝福彼此。临近毕业，我们一起再次回顾了2011年达特茅斯的毕业演讲，比起第一次在英语课上的生涩理解，我们有了更切身的体会，也以此赠与后辈和广大读者，正如柯南奥布莱恩所说，人生最美妙之处就在于一个人可以在失望中获得清醒，而在清醒中，则可以寻得激情和真正的灵感。十班的每一个人，尽管背景、经历、性格、喜好都各不相同，但是在这个集体中，我们以真诚相待，帮助彼此发现身上的闪光点，为每个人提供展现的舞台，也给与中肯的评价。



十班让每一个家人都得到尊重和支持，给与我们面对挫折的勇气和充满无限可能的未来。



RUN



四年中，我们实现了班级均分连续三年稳居全院19个班第一（共计30门课程位列全院第一，所有3+6课程均获全院第一），连续三年获评校先进团支部，学业、竞赛、党建、文体、生活各项评比数据都出类拔萃，凭借着共同的努力梦圆邦清，实现了入校最初的誓言。班内半数以上同学保研、出国，所有人都以优秀的成绩为这段奋斗岁月画上了圆满的句号。



而这四年带给我们的又何止荣耀的标签。圣诞趣味联欢会、春季骑行、足球场上挥洒汗水痛快踢球的少年英姿，乌龙的自助烧烤制造的奇葩笑料，都成为了大家最美好的青春记忆；承办留学周、德阳爱心行等富有意义的活动也使我们收获了新鲜的知识和独特的感悟。最值得珍惜的是生活中点点滴滴的相处，使我们懂得了彼此迁就包容，建立起深厚的情谊，即便终将别离，我们也永远是十班这个大家庭的一员。

也许比起未来人生画卷，大学四年仅是第一遍铅笔附上淡彩，但旅途至此已有何其多的曲折变幻，我们真的很幸运，在十班这个集体中，它充满温情和力量，使我们在互助中也充实了自己，于逆境中获得明晰和勇气，共同经历风霜雨雪，共同品尝彩虹的甜美，我们在这里共同创造了人生的第一个里程碑，它的名字叫做2011029100班。
